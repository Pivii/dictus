Build and install on device. Present the UAT checklist below to the user for verification.

What was built: Complete Phase 7 keyboard parity and visual polish — spacebar trackpad, adaptive accent key, universal haptics, emoji button, mic pill, recording pills, waveform rework, dynamic height, performance optimizations.

UAT checklist:

KBD-01 Spacebar Trackpad:
1. Open Notes, type a sentence
2. Long-press spacebar — feel distinct haptic, see greyed-out overlay
3. Drag finger left/right — cursor moves through text
4. Drag up/down — cursor jumps lines
5. Confirm NO haptics during drag movement
6. Release — overlay disappears
7. Short tap spacebar — inserts a space (no trackpad)

KBD-02 Adaptive Accent Key:
1. Look at AZERTY row 3 — key between N and delete shows apostrophe
2. Type "e" — accent key changes to e-acute
3. Type "a" — accent key changes to a-grave
4. Tap the accent key — inserts the displayed accent
5. Long-press the accent key after typing "e" — shows all e-accent variants
6. Switch to QWERTY — confirm NO accent key in row 3

KBD-03 Haptics on All Keys:
7. Tap each key type: letters, space, return, delete, shift, 123, #+=, emoji
8. Each tap should produce haptic feedback

KBD-04 Emoji Button:
9. Row 4 shows emoji face icon (not globe)
10. Tap emoji button — cycles to next keyboard

KBD-05 Apple Dictation Mic:
11. Note: system dictation mic below keyboard cannot be removed (iOS limitation)

KBD-06 Performance:
12. Type rapidly for 30 seconds — compare responsiveness to Apple keyboard
13. Haptic feedback should feel instant (no perceptible delay)

VIS-01 Mic Pill:
14. Toolbar mic button is pill-shaped (wider than tall), not circular

VIS-02 Recording Pills:
15. Tap mic to start recording
16. Cancel and validate are pill-shaped with glass effect

VIS-03 Waveform:
17. During recording: waveform animates with voice
18. Cover microphone (silence) — waveform becomes perfectly still
19. Stop recording — processing sinusoidal animation plays

Bonus checks:
- Keyboard height looks correct for your device
- Full Access banner Activer opens Dictus app
- Top-row key popups not clipped (or clipping is minimal)
