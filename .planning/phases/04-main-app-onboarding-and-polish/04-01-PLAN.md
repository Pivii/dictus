Define: `dictusBackground` (#0A1628 dark / appropriate light), `dictusAccent` (#3D7EFF), `dictusAccentHighlight` (#6BA3FF), `dictusSurface` (#161C2C dark / systemBackground light), `dictusRecording` (#EF4444), `dictusSuccess` (#22C55E).     **BrandWaveform.swift:** Create `BrandWaveform` view taking `energy: Float` (0.0-1.0) and optional `maxHeight: CGFloat` (default 80). Render 3 vertical bars in HStack(spacing: 8):
    - Left bar: white 45% opacity, base proportion 0.43
    - Center bar: LinearGradient from #6BA3FF to #2563EB, base proportion 1.0
    - Right bar: white 65% opacity, base proportion 0.64
    - Each bar height = baseHeight * (0.3 + energy * 0.7) * maxHeight
    - Bar width: 12pt, cornerRadius: 4.5pt
    - Animate with `.spring(response: 0.3, dampingFraction: 0.6)`
    - Include `@ScaledMetric` for barWidth so it scales with Dynamic Type

    **AnimatedMicButton.swift:** Create `AnimatedMicButton` view taking `status: DictationStatus` and `onTap: () -> Void`. States:
    - `.idle`/`.ready`: soft glow ring (dictusAccent at 0.3 opacity, pulsing 0.3-0.6 over 2s, repeatForever)
    - `.recording`: red pulse ring (dictusRecording at 0.5, scale 1.0-1.3 over 0.8s, repeatForever) + breathing scale on icon
    - `.transcribing`: blue shimmer sweep (overlay mask moving left-to-right, 1.5s repeat)
    - `.failed`: same as idle (reset state)
    - On transition from `.transcribing` to `.ready`: brief green flash (#22C55E, 0.3s fade)
    - Button disabled during recording and transcribing
    - Icon: `mic.fill` SF Symbol, 16pt medium weight

    **DictusTypography.swift:** Create `Font` extension:
    - `.dictusHeading` = `.system(.title, design: .rounded, weight: .bold)` — SF Pro Rounded
    - `.dictusSubheading` = `.system(.title3, design: .rounded, weight: .semibold)`
    - `.dictusBody` = `.system(.body)` — SF Pro Text (default)
    - `.dictusCaption` = `.system(.caption)`
    - All use system text styles so Dynamic Type works automatically
