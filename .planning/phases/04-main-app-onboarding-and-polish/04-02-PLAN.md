    - Animated logo: use BrandWaveform with energy transitioning from 0 to 0.5 on appear (spring animation, 1s delay)

    **MicPermissionPage.swift (Step 2):**
    - SF Symbol `mic.circle.fill` large icon in accent color
    - Title: "Microphone" in .dictusHeading
    - Explanation text: "Dictus a besoin du microphone pour transcrire votre voix. Vos enregistrements restent sur votre appareil."
    - [Autoriser le micro] button triggers `AVAudioSession.sharedInstance().requestRecordPermission`
    - Use async/await wrapper from research (withCheckedContinuation)
    - On grant: show checkmark, enable Next button, auto-advance after 0.5s
    - On deny: show warning message "Vous pouvez activer le micro plus tard dans Reglages" + still show Next button (don't block progress per research anti-pattern advice)
    - Track permission state with `@State private var permissionGranted: Bool?`

    **KeyboardSetupPage.swift (Step 3):**
    - SF Symbol `keyboard` large icon
    - Title: "Ajouter le clavier" in .dictusHeading
    - Two numbered instruction blocks:
      1. "Ouvrez Reglages > Dictus > Claviers > Ajouter un clavier" with [Ouvrir les Reglages] button using `UIApplication.shared.open(URL(string: UIApplication.openSettingsURLString)!)`
      2. "Activez l'Acces complet pour le microphone" with explanation "L'acces complet permet a Dictus d'enregistrer votre voix depuis le clavier"
    - Auto-detection: On `scenePhase` change to `.active`, call `isDictusKeyboardInstalled()` (from research — check UITextInputMode.activeInputModes for "com.pivi.dictus" identifier). If detected, show green checkmark + auto-advance.
    - Manual fallback: [J'ai ajoute le clavier] button always visible below auto-detection. Calls `onNext()`.
    - Don't block on detection — the manual button is the reliable path.

    **ModelDownloadPage.swift (Step 4):**
    - SF Symbol `arrow.down.circle.fill` large icon
    - Title: "Telecharger un modele" in .dictusHeading
    - Pre-select "small" model as recommended. Show model card with: name "Whisper Small", size "~500 Mo", quality "Bonne precision", note "Recommande pour la plupart des appareils"
    - [Telecharger] button starts download via ModelManager. Show ProgressView with percentage during download.
    - "Voir tous les modeles" link below that navigates to full ModelManagerView (NavigationLink)
    - On download complete: show checkmark, enable Next button, auto-advance after 0.5s
    - `@StateObject private var modelManager = ModelManager()` — or receive from environment if 04-01 set it up

    **TestRecordingPage.swift (Step 5):**
    - BrandWaveform component (from Design/) as main visual
    - Title: "Testez la dictee" in .dictusHeading
    - Prompt: "Dites quelque chose !" in .dictusBody
    - Large mic button (use AnimatedMicButton from Design/) to start test recording
    - When tapped: start recording via DictationCoordinator (access via @EnvironmentObject)
    - Show BrandWaveform animated with live energy during recording
    - After transcription completes: show result text in a card with .dictusGlass()
    - [Terminer] button becomes enabled after a successful transcription. Calls `onComplete()` which sets hasCompletedOnboarding = true in the binding.
    - If transcription fails: show error message + [Reessayer] button
